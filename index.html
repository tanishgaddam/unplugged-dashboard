Deploying Next.js to GitHub Pages

This repository contains the source code for a Next.js application.

To deploy this application to GitHub Pages, follow these steps:

  • Run the deployment script:
    npm run deploy
  • This script automatically builds the static application into the out/ folder and pushes it to the gh-pages branch.
  • Go to your repository settings on GitHub: Settings > Pages.
  • Under "Build and deployment", set the source to Deploy from a branch.
  • Check the branch selection and set it to gh-pages and the folder to /(root), then save.

(This index.html file ensures the root branch has an entry point, but the actual app lives on the gh-pages branch.)
